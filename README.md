# North Aster: Smart City Live Control System

A single-page smart city simulation with:

- live energy generation and consumption updates
- clickable city zones with floating control panels
- Arduino-style traffic light timers with manual override
- hospital, school, and residential load balancing
- day and night mode switching
- a compact city-systems shopping section in INR

## Run

Open [index.html](./index.html) in a browser.

## Mapbox

The app works immediately with the built-in local control grid.

To enable the tilted 3D Mapbox view:

1. Open the app.
2. Click `Mapbox Access`.
3. Paste a public Mapbox token.
4. Click `Connect Mapbox`.

The token is stored in local browser storage for convenience.
